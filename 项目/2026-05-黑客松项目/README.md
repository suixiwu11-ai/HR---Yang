## 项目概述

### 问题陈述
在AI时代，岗位概念日益流动化，传统的以岗位为标的的人效分析已无法准确衡量组织和个人的真实价值产出。如何重新定义人效评估维度，识别真正的高价值任务？

### 解决方案
通过划分任务包、任务域的形式，从工作任务场景出发重新定义组织。以「不可替代性」和「AI覆盖率」两个维度识别高价值任务，为受AI影响的白领创意工作行业（电商、咨询、游戏等）提供人效分析工具。

### 核心价值主张
不以岗位定人效，以任务价值重新定义AI时代的人效评估。
