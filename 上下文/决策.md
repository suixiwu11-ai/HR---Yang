# 决策日志

> 重大决策及理由记录。AI 在提出新建议前必须先读取此文件。
> ADR（架构决策记录）格式。

---

*暂无决策。当团队做出第一个重大选择时，在此添加记录。*

---

## 决策模板（需要时复制使用）

```markdown
## DEC-001: [决策标题]

- **日期**: YYYY-MM-DD
- **状态**: 提议中 / 已采纳 / 已废弃
- **参与者**: [涉及人员]
- **背景**: [为什么需要做这个决策]
- **考虑的选项**:
  - 选项 A: [描述] — [利弊]
  - 选项 B: [描述] — [利弊]
  - **选项 C（选中）**: [描述] — [选中原因]
- **决策**: [选择了什么]
- **理由**: [为什么选这个]
- **如果错了**: [我们如何知道，以及会怎么做]
```

---

*格式：复制 DEC-001 区块用于每个新决策，编号递增。*
